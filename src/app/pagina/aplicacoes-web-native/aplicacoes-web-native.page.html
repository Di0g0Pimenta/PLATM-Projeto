Aplicações Web Native

Uma aplicação web native é um tipo de software que é totalmente baseado na web e acessado através de um navegador. Ela não é instalada no dispositivo do usuário e pode ser acessada a partir de qualquer lugar com uma conexão à internet. Isso significa que uma aplicação web native não precisa se preocupar com diferenças entre sistemas operacionais ou dispositivos, pois ela é executada diretamente no navegador do usuário. No entanto, uma aplicação web native pode ter um desempenho um pouco inferior a uma aplicação nativa ou híbrida devido à dependência de uma conexão à internet e da capacidade do navegador em executar o código. Alguns exemplos de aplicações web native incluem o Google Docs e o Spotify Web Player.

Google Docs:

O Google Docs é uma aplicação web native é constituido usando tecnologias web, como HTML, CSS e JavaScript, mas é otimizado para funcionar como um aplicativo nativo. Isso significa que ele pode ser acessado através de um navegador da web, mas também oferece funcionalidades e experiências semelhantes a um aplicativo nativo, como offline access e notificações.

Características:

• Os usuários podem acessar seus documentos e apresentações em qualquer dispositivo, incluindo computadores, laptops, tablets e smartphones.

• É atualizado automaticamente, sem a necessidade de baixar novas versões.

• Google Docs permite a colaboração de varios usuarios trabalharem juntos em um documento tempo real.
